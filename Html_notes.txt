Main tags:
  <head> : comes at the top of the page to write main info about the page.
  <title> : written in the <head> and its content appears in the name of the tab.
  <body> : where all the content is shown.

Text tags:
  Writing tags:
    <h1> : headings for body and it comes in different sizes. from 1->6.
    <p> : to write a paragraph.

  Ways of writing:
    <b> : to make the text in between bold.
    <i> : to make the text in between italic.
    <s> : to make strikethrough. Cross the content.
    <sup> : to make the text in between raised up as 4th or 2 power 2.
    <sub> : to make the text in between lowered down as H2O.

  Breakers:
    <br /> : line breaker.
    <hr /> : to separate the two paragraphs with a line.

  Emphasis:
    <strong> : it indicates that the content is important. It makes it bold.
    <em> : it indicates that subtly changes the meaning of a sentence. It makes it italic.

  Quotes:
    <blockquote> : it's used for long quotes that take up an entire paragraph. The <p> is used inside.
    <q> :  it's used for shorter quotes that sit within a paragraph.

  Citation:
    <cite> : used to indicate where the citation is from.
    <dfn> :  used to indicate the defining instance of a new term.

  Abbreviations:
    <abbr title="text"> : used to write abbreviation and when hovering on it , it shows the whole definition.

  Author details:
    <address> : used to contain contact details for the author of the page.

  Insert and Delete:
    <ins> : used to show content that has been inserted into a document.
    <del> : used to show text that has been deleted from it.

Lists tags:
  Ordered Lists:
    <ol> : to create the ordered list.
    <il> : to add list items. It is put between the <ol></ol>.

  Unordered Lists:
    <ul> : to create the unordered list.
    <il> : to add list items. It is put between the <ul></ul>.

  Definition Lists:
    <d1> : used to create a definition list consists of terms and their definitions.
    <dt> : This is used to contain the term being defined.
    <dd> : This is used to contain the definition.

Links tags:
  <a href= "link"> : used to make the text in between a link.

Attributes:
   href : to write links.
   mailto : to create a link that addresses an email to a specified email address.
   target="_blank" : to open the link in a new window.
   id : to give a certain id to the tag so you can relate to it later.

Image Tags and Attributes:
  <img> : to add an image.
  src : to write the source of the image as a relative url.
  alt : to write description of the image.
  title : to write the title of the image.
  height = ""
  width = ""
  align = "right or left"
  <figure> : to contain images and their caption so that the two are associated.
  <figcaption> : to add a caption to an image.

Tables:
  <table> : to create a table.
  <tr> : to start a row.
  <td> : to enter cell.
  <th> : to represent the heading of a row or a column.
  scope="row or column" : to indicate whether it is a heading for a column or a row.
  colspan="number" : to stretch the entries in a table across more than one column.
  rowspan="number" : to stretch the entries in a table across more than one row.
  <thead> : to distinguish between entries in the head from the next item by a
  <tbody> : separating line
  <tfoot>
